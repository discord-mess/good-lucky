## 🛠️ Tính Năng Chính (Core Features)

Dự án tập trung vào việc phá nát máy:

1.  **Persistence (Startup Entry):** * Tự động đăng ký thực thi cùng hệ thống thông qua Schedulce tasks.
    * Mục tiêu: Duy trì hoạt động malware
2.  **Mass Payload Downloader:**
    * Tự động hóa việc tải xuống các tệp tin từ danh sách URL định sẵn.
    * Mục tiêu: Hấp diêm máy bằng 200+ con virus đảm bảo nát máy.
